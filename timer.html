Voice Commands

How to Use:
  1. Click the "Voice Control" button to enable voice commands
  2. Allow microphone access when prompted by your browser
  3. Say one of the commands below to control the timer
  4. The status indicator will show when commands are executed
Available Commands:
  • "reset" - Resets the timer to 0:00.0
  • "stop" - Pauses the timer (when running)
  • "start" or "resume" - Resumes the timer (when paused)
  • "settings" or "config" - Opens/closes the settings panel
Microphone Permissions:

If microphone access is denied, you'll need to allow it in your browser settings:

  • Chrome/Edge: Click the lock icon in the address bar → Site settings → Microphone → Allow
  • Safari: Safari → Settings → Websites → Microphone → Allow for this site
  • Firefox: Click the lock icon → Permissions → Microphone → Allow
Browser Support:

Voice commands work best in Chrome, Edge, and Safari. Firefox has limited support.

Intervals

Intervals Settings

Flash Mode: When enabled, the screen flashes when an interval is reached. When disabled, the timer text color changes and persists.
Auto Reset: Automatically resets the timer after the specified time. The reset time must be at least as long as your longest interval.
Intervals: Each interval has:
  • Time: Format as m:ss (e.g., 1:30 for 1 minute 30 seconds)
  • Color: The flash color or text color when this interval is reached
  • Popup Text: Optional message to display when the interval is reached
Apply & Reset: Saves your settings and resets the timer to start fresh with the new configuration.
